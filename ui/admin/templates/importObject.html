Regarding Conflicts

An import definition may contain one or more Tasks, and some of these Tasks may already exist in this environment.

The 'Conflicts' option defines a global directive for all conflicts. Tasks in the import file may also have a conflict resolution clause, but this global option takes precedence over the clauses in the file.

If 'Conflicts' is left blank, individual conflict resolution clauses will be honored.
